Image filtering

Image filtering is a fundamental technique in image processing and computer vision used to enhance or modify images by applying various mathematical operations to each pixel or neighborhood of pixels. These operations can include smoothing, sharpening, edge detection, noise reduction, and more. Image filtering plays a crucial role in tasks such as feature extraction, image enhancement, object recognition, and noise reduction.

Uses of filtering:

  • Enhance an image (denoise, resize, increase contrast, etc)
  • Extract information (texture, edges, interest points, etc)
  • Detect patterns (template matching)

Types of filtering:

  • Filters: Linear filters apply a linear transformation to the pixel values in the image or a neighborhood of pixels. Common linear filters include:
    • Smoothing Filters: Reduce image noise and detail by averaging pixel values within a local region. Gaussian blur and mean filter are examples of smoothing filters.
    • Sharpening Filters: Enhance image details and edges by accentuating high-frequency components. Examples include the Laplacian filter and the Sobel operator.
    • Edge Detection Filters: highlight edges and boundaries in images by detecting abrupt changes in pixel intensity.
  • Non-linear Filters: Non-linear filters apply operations that are not linear combinations of pixel values. They are often used for tasks such as noise reduction and image enhancement. Examples include:
    • Median Filter: Replaces each pixel value with the median value of its neighborhood, effectively reducing impulse noise (such as salt-and-pepper noise).
    • Bilateral Filter: Smooths images while preserving edges by considering both spatial and intensity differences between pixels.
    • Anisotropic Diffusion: Diffusion-based filtering method that selectively smooths image regions while preserving strong edges.
