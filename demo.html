Object Detection Demo

What is this page?

On this page you can get to see how CV can be used to detect objects in an image

To see, simply upload an image that is less than 3.5mb and click on the detect button. After a few seconds it will display some information about the image such as the object in it, and how many there are.

Try it!

(only works with png and jpg images)
